This project showcases a well-organized Linux file system built on Google Cloud Shell for Smart Nova Solutions, a technology company based in Kigali. The directory structure includes clearly arranged folders for company details, staff records, active and completed projects, system logs, and backup data.

Essential Linux commands such as mkdir, cat, cp, and >> were used to efficiently create directories, manage files, and generate server-related information. The entire project is version-controlled and published on GitHub to ensure proper documentation and professional presentation.
